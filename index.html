    MY PROJECTS

    QR Code Generator

    This project is a simple Python application that generates QR codes from any given URL. It utilizes the qrcode library to create QR codes, which can then be saved as image files or displayed directly.

    Heart Beat sensor

    This project involves creating a heartbeat monitoring system using an Arduino, a heartbeat sensor, and an LED screen. The system measures the heartbeat frequency and displays the real-time heart rate on an LED screen.

    Rock Paper scissiors Game

    This project is a simple command-line implementation of the classic Rock-Paper-Scissors game using Python. It allows a user to play against the computer, which randomly select its move.

    Photo to Pdf Converter

    This project converts multiple images into a single PDF file using Python. It uses the Pillow library for image processing and FPDF for PDF generation. Load images from a directory, convert them to PDF.

    Quiz Game

    This project is a console-based quiz game in C. It features multiple-choice questions, user input, score tracking, and feedback, demonstrating fundamental C programming concepts through an interactive application.
